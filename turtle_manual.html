Dette er en hjelpeside til turtle-verktøyet.

Bevegelse

Flytt og tegn

forward() | fd()

forward( distanse )

distanse
hvor langt turtlen skal flytte seg framover

forward: 

from turtle import *

forward(100)
color('red')   # kommer seinere
fd(50)         # alias

back() | bk() | backward()

distanse
hvor langt turtlen skal flytte seg bakover

backward: 

from turtle import *

backward(30)
color('blue')
back(50)
color('red')
bk(90)

right() | rt()

vinkel
hvor mange grader turtlen skal rotere mot høyre

right: 

from turtle import *

fd(30)
right(45)
fd(50)
rt(30)
fd(40)

goto() | setpos() | setposition()

x
et tall eller tallpar
y
et tall eller ingenting (hvis «tallpar» for x)

goto: 

from turtle import *

setpos(80, 80)   # gå til koordinatene x = 80 og y = 80
goto((100, -10)) # gå til koordinatparet (100, -10)
setposition(0, 0)

dot()

dot( størrelse=None, *farge )

størrelse
et heltall ≥ 1 (hvis gitt)
farge
en fargestreng (f.eks. "red" ) eller numerisk fargetrippel (f.eks. (0.2, 0.4, 0.55) )

dot: 

from turtle import *

dot(10, "red")
penup()        # kommer seinere
goto(90, 90)
dot()          # legg merke til at det kommer prikk også med 'penup()'
goto(0,0)

speed()

speed( \fart=None\ )

fart
et heltall i spennet 0 til 10, eller en fartsstreng

Sett turtlens fart til en heltallsverdi mellom 0 og 10. Fartsstrengen kobles til heltallsverdier som følger:

  • "fastest" : 0
  • "fast" : 10
  • "normal" : 6
  • "slow" : 3
  • "slowest" : 1

speed: 

from turtle import *

speed(1)
fd(100)
rt(90)
speed(4)
fd(100)
rt(90)
speed(8)
fd(100)
rt(90)
speed(0)
fd(100)
rt(90)

Pennen

Tegnetilstand

pendown() | pd() | down()

Sett pennen ned, slik at turtlen tegner om den beveger seg.

pendown: 

from turtle import *
penup()   # løfter pennen
goto(100, 100)
dot()
goto(30, 30)
pendown()
fd(90)

penup() | pu() | up()

Sett pennen ned, slik at turtlen tegner om den beveger seg.

penup: 

from turtle import *
goto(100, 100)
dot()
goto(30, 30)
penup()   # løfter pennen
fd(90)

pensize() | width()

pensize( bredde=None )

bredde
et positivt heltall.

Setter bredden til pennen til bredde. Hvis bredde ikke er gitt, returnerer funksjonen nåværende tykkelse.

pensize: 

from turtle import *

forward(50)
print(pensize())
pensize(5)
forward(50)

isdown()

Returnerer True hvis pennen er nede, False hvis den er oppe.

isdown: 

from turtle import *

penup()
fd(30)
if isdown() == False:
    dot()
pendown()
fd(50)
if isdown() == False:
    dot()
penup()
fd(30)

Fargekontroll

color()

color( *args )

Returner eller sett penn- og fyllfarge.

Denne tar alt fra 0 til 3 argumenter, som følger:

color()
Returner penn- og fyllfarge som et par av spesifikasjonsstrenger eller fargetupler.
color(fargestreng), color((r, g, b)), color(r, g, b)
Setter både penn- og fyllfarge etter tilsvarende format som pencolor().
color(fargestreng1, fargestreng2), color((r1, g1, b1), (r2, g2, b2))
Tilsvarende pencolor(fargestreng1) og fillcolor(fargestreng2).

For fargestreng-verdier, finner du mulige navn her.

color: 

from turtle import *

print(color())
fd(80)
color("deep pink")
fd(80)
